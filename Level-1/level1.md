# 用法

------

1. 克隆仓库
    `git clone https://github.com/SISUBEN/cyber-security-educate.git`

2. 打开cyber-security-educate > *Level-1* > application

   `cd cyber-security-educate/Level-1/applicaltion `  
3. 安装模块

	`pip install -r requirements.txt`
4. 启动app

	`flask run`
